## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic
Created a one-page infographic using Canva covering what digital literacy means, useful digital tools for students, and safe internet practices. The design aims to help batchmates quickly grasp the essentials of navigating the digital world responsibly.

### Task 2 – Student Digital Portfolio
Set up professional profiles on GitHub, LinkedIn, and Kaggle. Each platform serves a distinct purpose — GitHub for showcasing code and projects, LinkedIn for professional networking, and Kaggle for data science practice.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner-level challenge on HackerRank to build familiarity with competitive coding environments. Also built a 5-question Digital Literacy Awareness Quiz on Google Forms for use with batchmates.

🔗 **Google Form Quiz Link:** *(Add your Google Form link here)*

### Task 4 – Professional Email & Etiquette Guide
Drafted two professional emails and created a Social Media Do's and Don'ts checklist to promote responsible online communication habits among students.

### Task 5 – Cybercrime Awareness: UPI/Online Payment Fraud
Researched UPI and online payment fraud in India, wrote a detailed case study on how it occurs, and compiled an actionable prevention checklist for college students — including reporting resources such as **cybercrime.gov.in** and helpline **1930**.
