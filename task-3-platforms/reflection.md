# Task 3 – Coding & Collaboration Platforms

## Part A – HackerRank

I created an account on HackerRank and completed the **"Solve Me First"** challenge under the Problem Solving section. This challenge is designed for absolute beginners — it asks you to write a function that adds two numbers and returns the result. While the problem itself was straightforward, the experience of navigating the platform, reading the problem statement, writing code in the browser editor, and submitting for evaluation was genuinely new to me.

HackerRank also shows a badge and score after completion, which I have screenshotted and added to this folder.

## Reflection

This task introduced me to two different categories of online tools that I had heard of but never used seriously. HackerRank made it clear that coding practice platforms are structured, competitive, and very different from just writing code in a text editor. The immediate feedback you get when your solution is accepted or rejected is motivating in a way that a classroom exercise is not. I can see myself using it regularly to prepare for placement tests in later years.

The Google Forms exercise was equally useful. Building a quiz from scratch made me think carefully about how questions are worded, what answer options to provide, and how to make it fair and unambiguous. The fact that responses automatically populate a Google Sheet is something I had not appreciated before — it makes data collection extremely convenient for surveys, quizzes, and feedback forms. I can already think of several academic uses for this in group projects.
